# GSAP

Use this skill for GSAP animation tasks.

## Required workflow

1. Use GSAP plugin skills first. Select the most relevant skill:
   - Core: `gsap-core`
   - React/Next.js: `gsap-react`
   - Timelines: `gsap-timeline`
   - Scroll animations: `gsap-scrolltrigger`
   - Plugins: `gsap-plugins`
   - Performance: `gsap-performance`
   - Utilities: `gsap-utils`
2. For React/Next.js animation work, default to `gsap-react` and apply cleanup/scoping patterns.
3. For scroll-driven behavior, include `gsap-scrolltrigger` and validate reduced-motion handling.

## Capability check (required)

At the start of work, report:

```
GSAP capability check:
- Skills: available/missing
- Needed setup: install/config (if any)
```

If needed GSAP skills are unavailable, do not proceed silently. Tell the user what is missing and propose the exact skill/plugin setup needed.
