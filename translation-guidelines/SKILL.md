---
name: translation-guidelines
description: Use when the user asks to translate text from one language to another.
---
When translating text from one language to another:

- Translate meaning and intent, not word-for-word literal text.
- Produce natural, fluent text in the target language.
- Preserve the original tone (e.g., professional, friendly, marketing).
- Keep formatting, punctuation, and structure identical when possible.
- Do not modify code, variables, file names, URLs, or technical identifiers.
- If the text is inside HTML, JSX, JSON, or code, translate only the human-readable text and keep the structure unchanged.
- Maintain consistency for repeated terms across the project.
- If a phrase sounds unnatural in the target language, rewrite it to sound natural while preserving the original meaning.
- Do not add explanations or extra text.
- Output only the translated result.
